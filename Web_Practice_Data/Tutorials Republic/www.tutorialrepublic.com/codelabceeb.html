The database created in your browser using the Web SQL technology, and it contains sample tables that will help you practice with SQL.

You can restore the database at any time by clicking the Restore Database button.

Our SQL Editor uses Web SQL technology to store and access data on the client side.

WebSQL stores data locally, on the user's browsers. Therefore, each user can have their own database.

WebSQL is only supported in Chrome, Safari, and Opera. Open the URL of this page in any of these browsers to make the most of it.
