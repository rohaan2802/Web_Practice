HTML Color Picker

Create darker/lighter shades of any color.

Tints and Shades

Tints are lighter versions of the color that are made by mixing a color with white, whereas shades are darker versions of the color that are made by mixing a color with black. For example, pink is a tint of red, while maroon is a shade of red.

Note: Adding white to the color increases lightness, while adding black reduces lightness which results in darker color. Click on the color bars at the right hand side to further generate tints and shades for that color.
